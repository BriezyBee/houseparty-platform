# 🏠 House Party Platform

**Open‑source software for nonprofits and political campaigns.**  
AGPL licensed. Built by and for community organizers.

## What is this?

This platform helps small organizations manage:
- ✅ Volunteer shift sign‑up and waitlists
- ✅ Training reminders with attachments
- ✅ Endorsement announcements
- ✅ Mutual aid logistics
- ✅ Candidate education tracking

## Current status

🚧 **Early development** – First module (volunteer scheduling) is in progress.

## Tech stack

- Database: PostgreSQL
- Backend: Node.js + Express
- Frontend: React + refine
- Deployment: Docker
